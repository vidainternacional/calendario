# Calendario Vida

Aplicación de calendario para la gestión de eventos e iglesias — Vida Internacional.

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Supabase (PostgreSQL + Auth)
- TailwindCSS

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.
